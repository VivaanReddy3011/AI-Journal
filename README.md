# Student Journal Demo

A fast, free, local-first demo that lets a student write journal entries, analyzes the emotional tone, and shows supportive feedback plus a simple mood dashboard.

## What it includes

- Journal input
- Local emotion analysis
- Supportive feedback and advice
- Mood dashboard with charts
- Entry history saved in the browser
- Safety banner for urgent language

## How to run

### Option 1: Open directly

Open `index.html` in your browser.

### Option 2: Use a local server

If you want a localhost URL, run one of these from this folder:

```powershell
npm start
```

or

```powershell
python -m http.server 8000
```

Then open:

```text
http://127.0.0.1:3000
```

## Notes

- Everything stays local in your browser.
- Data is stored in `localStorage`.
- This is a demo, not a clinical mental health tool.
